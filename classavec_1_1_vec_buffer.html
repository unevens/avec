Detailed Description

template<class Vec>
class avec::VecBuffer< Vec >

A wrapper on aligned_vector used to manage interleaved memory that can be mapped to simd vector objects.

Template Parameters
Vecthe simd vector object type that the VecBuffer can be mapped with.
